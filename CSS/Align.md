## For container:   
```  
display: flex;
align-items: center;
justify-content: center;
```  
## For text in the container:  
```  
text-align: center;
```  
## For inner container (eg: 'span' elements)  
span elements are in a line by default
```  
display: inline-flex;
align-items: center;
```  
to put span elements in a vertical line:  
```  
display: block;  
```
